## Things

\> go right
< go left
. print as ascii
(any number) + adds any number
(any number) - subtracts any number
@ (any number) sets to specific number
$ (cmd) runs cmd only if current is 1
0 resets current
:(label name) define label
^(label name) goto label
~ returns to last goto cmd
